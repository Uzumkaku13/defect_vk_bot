# VK Bot — дефектная ведомость в PDF

Бот для сообщества VK, который:
- ведет пошаговый сбор дефектной ведомости;
- сохраняет черновик в SQLite и не теряет прогресс после перезапуска;
- принимает дефекты с фотографиями;
- формирует итоговый PDF через `reportlab` в деловом табличном виде;
- использует **Times New Roman** при наличии системных шрифтов Windows и корректно печатает кириллицу;
- вставляет логотип компании сверху документа, а если файла нет — печатает текстовую заглушку `ООО "Техноком"`;
- раскладывает фото в PDF так: **вертикальные по две рядом**, **горизонтальные на всю ширину блока**;
- сжимает фотографии перед сохранением и дополнительно включает `pageCompression=1` для PDF;
- отправляет готовый PDF обратно в чат VK;
- присваивает номер ведомости в формате `№ ГОД-0000001`.

## Быстрый запуск

1. Создайте `.env` на основе `.env.example`
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Инициализируйте БД:
   ```bash
   python init_db.py
   ```
4. Запустите бота:
   ```bash
   python bot.py
   ```

## Пример `.env`

```env
VK_TOKEN=your_vk_group_token
GROUP_ID=123456789
DB_PATH=bot_data.sqlite3
DOCS_DIR=generated_docs
MEDIA_DIR=media
LOGO_PATH=assets/tehnokom_placeholder.png
LOG_LEVEL=INFO
```

## Команды в меню

- `🆕 Новая ведомость`
- `📂 Продолжить черновик`
- `✅ Завершить и получить PDF`
- `📜 Мои ведомости`
- `📊 Статистика`
- `❌ Отменить текущую`

## Повторная отправка файла

Отправьте в чат:
```text
файл 2026-0000001
```
или старый короткий вариант по ID:
```text
файл 12
```

## Важные замечания

- Для загрузки готового PDF в чат у токена сообщества должны быть права на сообщения и документы.
- На Windows бот подключит системные шрифты **Times New Roman** (`times.ttf`, `timesbd.ttf` и т.д.).
- Если Times New Roman на конкретной машине недоступен, будет использован резервный Unicode-шрифт с поддержкой кириллицы.
- Фото дефектов бот сохраняет локально сразу при получении, затем вставляет в PDF по локальному пути.
- Для уменьшения размера итогового файла все загружаемые фото автоматически приводятся к JPEG и сжимаются.
